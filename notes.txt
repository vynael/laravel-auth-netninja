# 📝 Catatan Git & Setup Laravel Project

## 📁 Membuat Folder Menjadi Repository Git

# Inisialisasi repository Git
git init

# Menambahkan semua file ke staging area
git add .

# (Opsional) Tambahkan file tertentu saja
git add nama_file.php

# Commit perubahan pertama
git commit -m "first commit"

# Tambahkan remote repository (GitHub)
git remote add origin https://github.com/vynael/laravel-auth-netninja.git

# Rename branch ke 'main' (opsional, tergantung GitHub)
git branch -M main

# Push ke GitHub
git push -u origin main

------------------------------------

## ⬇️ Clone Repository (di komputer lain / rekan tim)

git clone https://github.com/vynael/laravel-auth-netninja.git
cd laravel-auth-netninja

---

## 🔄 Sinkronisasi Repo Lokal dengan Remote

# Ambil update terbaru dari remote
git pull origin main

---

## ⚙️ Setup Laravel Setelah Clone

# Install dependency PHP
composer install

# Install dependency frontend (JS/CSS)
npm install

# Duplikasi file environment
cp .env.example .env

# Generate app key Laravel
php artisan key:generate

# (Opsional) Jalankan migrasi database
php artisan migrate

---
